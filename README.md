# Greenesia Android
Greenesia is a platform for building a green environment. You can join many event to threat our nature, plat tree, or any social activity. Collect point when you join event, and claim cool reward.

## Tech Used:
* Kotlin
* Support Library
* Rx Java & Rx Android
* Retrofit
* Moshi
* Picasso
* ZXing
* Dagger 2
* etc.
